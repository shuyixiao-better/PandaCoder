## 快速更新版本

### 1. 修改版本配置

编辑 `gradle.properties` 文件：

```properties
pluginVersion=1.2.0
versionType=正式版本
releaseDate=2025-01-01
currentFeatures=新功能描述
```

### 2. 运行更新脚本

```bash
python update-version.py
```

### 3. 重新构建

```bash
gradlew clean build
```
